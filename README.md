# xfw 个人博客

免费静态博客，Gitee Pages 托管，国内访问快。

## 文件结构

```
blog/
├── index.html          # 首页
├── blog.html          # 博客文章列表
├── about.html         # 关于页面
├── style.css          # 样式
├── posts/            # 文章目录
│   └── 欢迎来到我的博客.html
└── README.md         # 本文件
```

## 部署到 Gitee Pages（5 分钟搞定）

### 第一步：注册 / 登录 Gitee
- 打开 https://gitee.com
- 注册账号（需要手机号，免费）
- 登录后点击右上角「+」→「新建仓库」

### 第二步：创建仓库
- 仓库名称：随便起，比如 `blog`、`xfw-blog`
- 是否公开：公开（否则 Gitee Pages 要收费）
- 勾选「使用 README 初始化仓库」
- 点击「创建」

### 第三步：上传文件
- 进入仓库页面，点击「文件」→「上传文件」
- 把 `blog/` 目录下**所有文件**拖进去（包括 `posts/` 子目录）
- 提交信息随便写，比如「初始化博客」
- 点击「提交」

### 第四步：开启 Gitee Pages
- 进入仓库，点击上方「服务」→「Gitee Pages」
- 分支选 `master`
- 部署目录选 `/`（根目录）
- 点击「启动」
- 等待 1-2 分钟，你会得到一个地址：`https://你的用户名.gitee.io/仓库名`

### 完成！
用那个地址就能访问你的博客了，国内访问速度很快。

## 以后怎么写新文章

1. 在 `posts/` 目录下新建一个 `.html` 文件（复制 `欢迎来到我的博客.html` 改改）
2. 改标题、日期、内容
3. 在 `blog.html` 和 `index.html` 的文章列表里加上新文章的链接
4. 上传到 Gitee（网页上直接「新建文件」或「上传文件」）

如果不想手动改 HTML，可以用任意文本编辑器写，让我帮你转成网页。
